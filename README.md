# CalendarExplorer

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-CLI-orange.svg?logo=terminal)](https://en.wikipedia.org/wiki/Command-line_interface)

Простая программа для изучения работы с Calendar API в Kotlin и форматирования дат в разных локалях.

## О проекте

Программа демонстрирует различные способы работы с датами и временем через класс Calendar в Kotlin. Показывает получение и форматирование компонентов даты в разных локалях.

**Основной функционал:**
- Установка и получение года
- Извлечение часов и минут
- Форматирование месяцев в коротком и полном формате
- Работа с разными локалями (включая китайскую)
- Демонстрация методов Calendar API

## Технологии

- Kotlin
- Java Calendar API
- Локализация (Locale)
- Форматирование дат и времени
- Многовариантное отображение месяцев
